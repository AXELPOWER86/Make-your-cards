📝 Come creare una carta
  • Scrivi il testo della carta nel campo apposito: si adatterà automaticamente alle dimensioni della carta
  • Clicca ⬇ Scarica PNG per salvare la carta nella tua galleria
  • Clicca 💾 Salva nel mazzo per aggiungere la carta al tuo mazzo personalizzato
  • Clicca 🗑 Cancella testo per pulire il campo di testo
  • Clicca 🗑 Cancella tutto per eliminare tutte le carte del mazzo
  • Clicca 💾 Backup Mazzo per scaricare tutte le carte (JSON + PNG)
  • Clicca 📂 Importa backup per ripristinare un mazzo da file JSON
  • Clicca sulla carta o su ✏️ per modificarla, clicca una seconda volta per deselezionarla
  • Clicca 🌐 Condividi in community per condividere la carta selezionata
  • Clicca 🗑 accanto alla carta per eliminarla dal mazzo
  • ☁️ Accedi con Google per il backup automatico nel cloud (necessario anche per condividere)
  • Opzionale: carica uno sfondo personalizzato tramite 📂 Carica sfondo carta
🔒 Informativa privacy e Credits

Porky's Game® — Editor Carte non raccoglie dati per analisi, statistiche o attività di tracciamento.

I dati del mazzo personalizzato vengono salvati esclusivamente in locale sul tuo dispositivo tramite localForage. Se effettui l'accesso con Google, una copia di backup viene sincronizzata sul tuo account Firebase personale e non è accessibile ad altri utenti.

Solo in caso di condivisione volontaria con la community, gli altri utenti potranno vedere le tue carte.
